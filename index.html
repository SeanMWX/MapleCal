MapleCal 战斗力计算器

纯静态页面，输入数据即可计算战斗力、面板与输出，并查看各属性增量的输出提升。

属性 (Attributes)

攻击 (Attack)

伤害 (Damage)

其他 (Other)

战斗力 ?
属性 = floor((主属性基础值 - 主属性技能) * (1 + 主属性%) ) + 主属性非%加成
副属性 = floor((副属性基础值 - 副属性技能) * (1 + 副属性%) ) + 副属性非%加成
属性总和 = (主属性 * 4 + 副属性) / 100
攻击 = floor((攻击力基础值 - 攻击力技能 + 皇后祝福 + weapon_fix) * (1 + 攻击力%))
伤害 = (1 + 伤害% - 伤害技能%) + (Boss伤% - Boss伤技能%)
暴击 = (1.35 + 暴伤% - 暴伤技能%)
伤害乘区 = 伤害 * 暴击
战斗力 = 属性总和 * 攻击 * 伤害乘区 * (1 + 创世武器最终伤害%) * (1 + 怪怪卡最终伤害%)

-
新面板。

面板 ?
主属性 = floor(主属性基础值 * (1 + 主属性%) + 主属性非%加成)
副属性 = floor(副属性基础值 * (1 + 副属性%) + 副属性非%加成)
属性总和 = (主属性 * 4 + 副属性) / 100
攻击 = floor(攻击力基础值 * (1 + 攻击力%) + 攻击力非%加成)
面板伤害 = (1 + 伤害%) * (1 + 最终伤害%)
面板 = 属性总和 * 武器系数 * 攻击 * 面板伤害

-
旧面板。

输出 ?
主属性 = floor(主属性基础值 * (1 + 主属性%) + 主属性非%加成)
副属性 = floor(副属性基础值 * (1 + 副属性%) + 副属性非%加成)
属性总和 = (主属性 * 4 + 副属性) / 100
攻击 = floor(攻击力基础值 * (1 + 攻击力%) + 攻击力非%加成)
最终伤害 = (1 + 最终伤害%)
伤害加成 = (1 + 伤害% + Boss伤%)
暴击加成 = (1 + 暴伤%)
输出 = 属性总和 * 武器系数 * 攻击 * 最终伤害 * 伤害加成 * 暴击加成

-
